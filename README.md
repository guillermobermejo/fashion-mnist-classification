# Fashion-MNIST CNN (Beginner)

A beginner computer vision project using PyTorch.

## Goal
Train a simple CNN to classify Fashion-MNIST clothing images into 10 categories.

## Setup (Windows / PowerShell) (Warnings: Ctr + Shift + P -> Select interpreter -> .venv)
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python train.py
